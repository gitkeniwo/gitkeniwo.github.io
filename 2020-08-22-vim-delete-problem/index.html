mac vim中有时delete键在insert模式下无法向左删除

ケニヲ
2020-08-22
Tech
shell, vim

mac vim中启动了某种compatible模式，为了与vi兼容，而无法删去某些特定字符。

只需在~/.vimrc 中加入一行 set backspace=2 即可

实测完美解决，目前没出问题。

参考：https://www.smslit.top/2016/11/27/vim-backspace-invalid/
