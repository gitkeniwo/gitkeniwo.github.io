禅与自行车维修艺术 - 在尼德兰给自行车前轮换内胎

ケニヲ
2025-12-31
Tweets › Lifehacks
netherlands

住尼德兰最糟心的是住在市中心，又没有方便的bikeshed。如果停外面的话，你的车活不过一个月就会被扎被损坏，即便是不起眼的破车。
OP住的h2s的CT的bikeshed在负一楼，只有两条路通向该bikeshed：1）坐唯一一部能去负一楼的电梯下负一楼，然后经过三道门 2）通过一个地下停车场，刷卡进入，并在地下绕我们楼一圈。这样看来，如果等电梯还不知道有多挤，基本上取车就需要+5min，实在麻烦死了。

所以目前看来，最合适的车是可以停进屋的折叠自行车。但是此类车一般都是250+欧元，如果从墙内托运过来，还要担心尺寸以及轮胎维修无零部件的问题。

十二月中旬OP去微信群里找了个大婶50欧收了一辆脚刹的Omafiets（荷兰语直译过来就是「老妈车」）。OP还在沾沾自喜车况很好，结果不到一周车的前轮就废了。OP高度怀疑本车是西方大城市市中心十分常见的小混混vandalism的受害者。

那么，如果去土耳其街找中东修车铺换个胎呢？车推去一问，不好意思，45……

今天是跨年夜，窝终于下定决心，一定要把车修了，把车的前轮轮胎换了。

Bicycle 101: 修车原理

  1. 自行车轮胎中可以卸下来的结构是外胎 + 内胎 + 气嘴 + 胎垫。其中，外胎的横截面是个只有一半（180度）的圆环，内胎横截面则是完整圆环；内胎才是能被充气充起来的那个，但是比较脆弱，所以在内胎外面加了一层半包裹的外胎作为保护。
  2. 自行车除去胶皮轮胎后的铁架叫轮毂(hub)，最外面的那层可以上轮胎铁圈叫轮辋(rim)，以及径向的钢条叫辐条(spoke)。
  3. 内胎不光是一层胶皮，还有气嘴。气嘴可以通过自行车轮毂上一个孔安插进去。
  4. 车打不进去气一般是内胎+气嘴有问题。首先确定轮胎充不进去气的原因：我的是气嘴(valve/ventiel)与内胎(inner tube/binnenband)连接处断裂，打气时就是松的，而不是先打了气后来轮胎再瘪了。

Tools

准备以下工具。由于荷兰至少还是名义上的「自行车王国」，所以修车工具都非常成体系和标准化，相对容易购买：

fig. 所有工具

  • 开口扳手，用来卸下固定车胎的六角螺丝。
    • 英文叫Spanner或者Open-end Wrench，荷兰语为Steeksleutel
    • 我买的Action的Werckmann这个牌子的Adjustable Joint Pliers，荷兰语商品名为Werckmann waterpomptang 245mm，€3.99
    • 按理说也可以买外六角螺丝刀（梅花扳手, ring wrench），我也买了Action的Werckmann ringsteeksleutelset 6-delig，€3.99 但没用上。
  • Action的Tyre Repair Kit，重点是里面的撬胎棒，Bandafnemer (荷兰语 af+nemen = remove) 或 Bandenlichter 如果不急的，可以去Temu/Aliexpress自己凑单买个便宜的。
  • 内胎英文叫Inner Tube，荷兰语为Binnenband。 买内胎需要选对气嘴型号+内胎型号。
    • 观察自己的轮胎，如果实在不明显可以打手电筒反光，拍照拉曲线拉对比度等，一般会标注轮胎的型号，荷兰的外胎一般会标注 [ETRTO] [British] [French] 三种规格，最简单易懂的是 British 英制规格, outer diameter x height x width，一般如同 26 x 1 ⅜这样，意思就是26’’(英寸) + 胎宽胎高。我的胎是 [26 x 1 ⅜] [37-590] [650 x 35A]，其中37-590是ETRTO 标准，是EU制定的最准确的标准，650 x 35A是法制。
    • 轮胎还需要注意自己气门(Valve)的型号，荷兰老款车常用英式/荷式的Dunlop，此外著名的还有法嘴Presc，和美嘴Schräder。
    • 大众款山地车可以直接迪卡侬买，其他老式车如Omafiets就去著名荷兰建材超市Praxis买，可以网上下单然后预约取货，取货48小时内都可以。

fig. OP的轮胎型号，肉眼无法看清，拉了对比度和曲线才读出来

开干

万事俱备直接开干

A. 首先，如果是尼德兰的冬天，你需要做的是把车推到一个温暖的地方，例如你的房间里，要不然你会被冻死。

B. 第二步，将自行车倒过来，方便卸轮胎（literally Swapfiets (荷兰一个著名的租车品牌))

C. 用活动扳手卸下你的轮胎。直接上活动扳手反方向拧，我认为还是比较轻松的。因为本帖只讨论前轮，前轮特别简单，两头各有一个六角螺丝，和两个铁片

D. 卸载外胎。

D.0 放气。一定要先放气。如果不是美嘴，请直接插下气门所有可拆的部件。

fig. 放气

D.1 理解撬胎棒的构造。撬胎棒一侧是一个类似勺子的结构可以把外胎的胎唇挖出来，另一侧是一个钩子，可以系在辐条上。理想的撬胎模式如下图。

fig. 撬出来后钩到辐条上

D.2 OP非常不幸的遇到了外胎轮唇过紧的问题。因为之前没仔细看，直接上蛮力，直接折断了一根塑料撬胎棒，都没有撬成功。
很多人也介绍了给外胎做按摩的技巧，把两侧胎唇按到轮辋的中心，因为中心是凹下去的，就可以让胎唇更送。

D.3 小技巧：因为反正要换内胎，直接把内胎剪断，扯出来即可。OP只能这么做了……

D.4 OP又很不幸的剪断了胎垫（rim tape, 荷兰语velglint）, OP并不知道有这个结构的存在，以为胎垫是损坏的内胎。需要再氪金€3.49买一根胎垫……

fig. 没有胎垫的轮毂，内胎遇到这个铁制轮辋直接就寄了

fig. 花重金重新购置Velglint。有时间一定要去Temu慢慢买🙂

OP检查了下内胎，发现漏气的原因竟然是这个胎被前任车主补过，而补过的地方坏了。

E. 安装内胎。需要给内胎打一点点气，让它大致有个形状。

E.1 先装一侧外胎。
E.2 然后装内胎。1) 先使气嘴对准轮辋上的气嘴孔。 2) 把剩余部分塞进去。让外胎像个袋子兜住内胎。 3) 不要让外胎胎唇压着内胎，不然充满气直接送走. （忘了拍照了）
E.3 把另一侧外胎装回来。用撬胎棒撬回最后的一点点。注意：装外胎时，需要反向使用撬胎棒。
E.4 再次检查不要让外胎胎唇压着内胎，特别是气嘴对应的那一段！调整气嘴使其对着外胎肚兜的中心，不要被外胎夹住了。

fig. 满血复活

F. 打满气。把轮胎安装回车架上。

fig. 快乐的扭上螺丝

后轮

Lucky that op这次只用换前轮。后轮是十分恐怖的，而且脚刹车大概还要考虑如何把后轮刹车结构一起拆下来。

fig. 看看这个可怕的结构和工作量

所以我又查了下换后胎的视频（关键词：荷兰语omafiets binnenband vervangen），例如这个老哥基本上没动链条的那一侧，只拆了好拆的一侧，未卸载整个轮子，直接撬胎把内胎扯出来换了。估计这是一种思路。(let’s hope we never have to deal with smth like that…)

最后一些毒力撕烤

The real motorcycle you’re working on is a cycle called yourself. —ZAMM

修车确实是一个点亮黑暗地图的过程。在下决心购置工具，OP处于天天靠走路的摆烂状态，不愿掏这份钱。不愿去了解自己修车的可能性。

ZAMM告诉我们，对于摩托车也好、自行车也好，这样外在的机器的态度，很大程度上也就是我们自己对于内在的自我的态度。
按照ZAMM的说法，摩托车/自行车的构建遵循一套用钢铁构建的、极其严密的逻辑系统。当你购买、租赁、请人修理好一辆完整的车，只想要体验与感受兜风骑行的乐趣时，ZAMM说你本质上是Romantic的；对于更古典派（Classic）的人来说，车是beyond其外在的，是零件、机械，是一整套概念原理。

自行车轮胎能充气是靠内胎驱动的，而外胎仅是轮胎的浅显外在。在动手修车之前谁知道轮胎是这样组装起来的？它是在荷兰生活每日都离不开的事物，但OP对它知之甚少。
生活中其它不起眼的、但窝萌习以为常的概念，也大多有如此不为人知的一面。人习惯于黑盒按照黑盒的方式运作，黑盒之内就是我们理性的边界。
维修摩托车/自行车能让人快速达到心流状态，完全沉浸在工作中。大脑前额叶皮层会释放大量多巴胺，让人感知敏锐、深度专注，DMN被抑制，导致「自我」的消失，修理者与机器合二为一。
ZAMM所说的禅是可以推广的，也就是说，在静心静气的去关注、去理解、去探索的过程中，窝萌也可以达到那种物我合一的精神状态。

某种意义上讲，摩托车/自行车是外在的现代工业文明的一种具象，而维修它理解它掌控它是窝萌保持不被技术异化、并与之和谐相处的一种切入途径。

修自行车已被OP列为在荷兰生活的T0级技能。

（悲惨的）后续

当晚，内胎打满气后，又爆了。因为我停在家里，三点钟一声惊天地的巨响让op怀疑人生，op还以为是什么暖气片爆了，或者外面玩Cobra 8呢，起床排查一圈房间里没发现原因。。。今早起来一摸车胎又瘪了。

总结下可能的原因：
1）内胎装进去的时候被外胎胎唇压了，尽管已经检查过
2）轮毂生锈，然后胎垫不够宽
3）外胎装反？

接下来准备尝试：
1）先取下内胎，检查爆胎原因是被夹了还是因为胎垫不够还
2）给轮毂除锈。a.用除锈剂 b.用砂纸或者钢丝刷
3）换更宽的胎垫or真空胎垫
4）装上内胎时，先不打满气
5）买一个能测气压的电动pump。
